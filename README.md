# YouTube Sans for Stripo

1. Create a public GitHub repository.
2. Upload all files from this folder.
3. Enable GitHub Pages (Settings > Pages > Deploy from branch > main > /(root)).
4. Your CSS URL will be:
https://YOUR_USERNAME.github.io/REPOSITORY_NAME/youtube-sans.css
5. Paste that URL into Stripo > Project Settings > Font Management > Add Custom Font.
Font family: YouTube Sans
